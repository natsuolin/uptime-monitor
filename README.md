🌟 Website Monitoring Project 🌟
This project is designed to monitor websites and provide real-time updates on their status. It uses a combination of Express.js, SQLite3, and cron jobs to achieve this functionality. The project allows users to add, remove, and view websites, as well as view the log history for each site. The monitoring job runs periodically to check the status of each website and update the database accordingly.

🚀 Features
Add, remove, and view websites
View log history for each site
Real-time updates on website status
Periodic monitoring job to check website status
API routes for interacting with the database
🛠️ Tech Stack
Frontend: None
Backend: Express.js
Database: SQLite3
AI Tools: None
Build Tools: None
Other: cron jobs
📦 Installation
To get started with the project, follow these steps:

Prerequisites
Node.js installed on your system
SQLite3 installed on your system
Installation
Clone the repository using git clone
Navigate to the project directory using cd
Install the dependencies using npm install

Start the server using node src/server.js
💻 Usage
To use the project, follow these steps:

Start the server using node src/server.js
Open a web browser and navigate to http://localhost:3000
Use the API routes to interact with the database
